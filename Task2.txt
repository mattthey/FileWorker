	Задача №2 - FileWorker - ООП

	Необходимо разработать класс для работы с файлами - FileWorker:

1. public FileWorker(String path) - конструктор принимает один строковый параметр - путь до каталога с файлами.
Пример: "D:\path". Параметр неизменяем.

2. Поле boolean isRecursive, get и set - устанавлвиает режим работы: рекурсивный или нет.
То есть применить действие к подкаталогам или нет.

3. Метод void execute(IExecutable command) - метод, который принимает объект
и спомощью этого объекта производит некое действие над каталогм, заданным параметром path в конструкторе.
Что именно происходит с файлами - определяется конкретной реализацией интерфейса IExecutable.


Интерфейс IExecutable содержит один метод:

1. void proceess(File f) - принимает на вход файл, над которым необходимо произвести действие. 

То есть, FileWorker просто перебирает файлы либо плоско, либо как дерево в зависимости от параметра isRecursive.
И для каждого файла вызывает метод proceess из объекта command.

Для примера необходимо реализовать Md5Executor implements IExecutable.
Соотвественно метод proccess вычисляет md5-хэш файла.
Результат выводить в файл в виде <путь до файла относительно path>: <хэш>.


Также необходимо в рамках задачи понять, что являеется хэшом каталога. Так же выводить в файл.
*. Реализовать другой алгоритм хэширования.
**. Реализовать механизм возврата результата, предусматривающий результат разного типа.
